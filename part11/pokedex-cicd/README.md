## Quick Start:

**Install Dependencies:**

```bash
npm install
```

**Build the Frontend:**

```bash
npm run build
```

**Run (Development):**

```bash
npm run start
```

**Run (Production):**

```bash
npm run start-prod
```

**Run Linting:**

```bash
npm run eslint
```

**Run Tests:**

```bash
npm run test
```

**Run End-to-End Tests:**

```bash
npm run test:e2e
```
