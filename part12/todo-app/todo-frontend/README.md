# Todo Frontend (React application)

This application is created with [Vite](https://vitest.dev/).

---
---

## Quick Start:

Install Dependencies:

```bash
npm install
```

Run Server (Dev):

```bash
npm run dev
```

Build Static Files (Prod):

```bash
npm run build
```

---
---

## Environment variables

- `VITE_BACKEND_URL`: Set backend URL.
