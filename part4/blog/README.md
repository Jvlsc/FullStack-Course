## Quick Start:

**Install dependencies:**

```bash
npm install
```

**Run the project (production mode):**

```bash
npm run start
```

**Run the project (development mode):**

```bash
npm run dev
```

**Run tests:**

```bash
npm run test
```

**Run linting (All):**

```bash
npm run lint:all
```

**Run linting (API):**

```bash
npm run lint:api
```

**Run linting (Blog Model):**

```bash
npm run lint:model
```

**Run linting (List Helper):**

```bash
npm run lint:list
```
