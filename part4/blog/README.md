## Quick Start:

**Install dependencies:**

```bash
npm install
```

**Run the project (production mode):**

```bash
npm run start
```

**Run the project (development mode):**

```bash
npm run dev
```

**Run tests:**

```bash
npm run test
```

**Run linting (All):**

```bash
npm run lint:all
```

**Run linting (API):**

```bash
npm run lint:api
```

**Run linting (List):**

```bash
npm run lint:list
```
