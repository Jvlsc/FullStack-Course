## Quick Start:

**Install Dependencies:**

```bash
npm install
```

**Run the Project (Production Mode):**

```bash
npm run start
```

**Run the Project (Testing Mode):**

```bash
npm run start:test
```

**Run the Project (Development Mode):**

```bash
npm run dev
```

**Run Tests (All):**

```bash
npm run test:all
```

**Run Tests (All APIs):**

```bash
npm run test:api
```

**Run Tests (Blogs API):**

```bash
npm run test:api:blogs
```

**Run Tests (Users API):**

```bash
npm run test:api:users
```

**Run Tests (All Models):**

```bash
npm run test:models
```

**Run Tests (Blog Model):**

```bash
npm run test:model:blog
```

**Run Tests (User Model):**

```bash
npm run test:model:user
```

**Run Tests (List Helper):**

```bash
npm run test:list
```

**Run Linting:**

```bash
npm run lint
```
