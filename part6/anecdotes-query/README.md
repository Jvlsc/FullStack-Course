## Quick Start:

**Install Dependencies:**

```bash
npm install
```

**Run the Project:**

```bash
npm run dev
```

**Run the Backend:**

```bash
npm run server
```

**Build the Project:**

```bash
npm run build
```

**Run Linting:**

```bash
npm run lint
```
