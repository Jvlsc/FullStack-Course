## Quick Start:

**Install Dependencies:**

```bash
npm install
```

**Run the Project:**

```bash
npm run dev
```

**Build the Project:**

```bash
npm run build
```

**Run Tests:**

```bash
npm run test
```

**Run Linting:**

```bash
npm run lint
```
