## Quick Start:

```bash
npm install
```

**Transpile the Backend:**

```bash
npm run tsc
```

**Run the Backend (Production):**

```bash
npm run start
```

**Run the Backend (Development):**

```bash
npm run dev
```

**Run Linting:**

```bash
npm run lint
```
