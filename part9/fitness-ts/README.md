---
---

## Quick Start:

**Install Dependencies:**

```bash
npm install
```

**Run the Backend:**

```bash
npm run start
```

**Run the Backend (Dev):**

```bash
npm run dev
```

**Run Script - BMI Calculator:**

```bash
npm run calculateBmi
```

**Run Script - Exercise Calculator:**

```bash
npm run calculateExercises
```

**Run Linting:**

```bash
npm run lint
```
